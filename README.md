# kiloton

This is the kiloton app

##Instalation

1. ```git clone git@github.com:USERNAME/kiloton.git``` (After getting authentication of host).

2. Install cocoapods: ```sudo gem install cocoapods``` ([RubyGems](https://rubygems.org/pages/download) required).

3. run: ```pod install```

4. open `kiloton.xcworkspace`
